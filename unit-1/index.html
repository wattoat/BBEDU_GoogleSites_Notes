MM1.1

Laws of Indices

Indices give us a compact notation for repeated multiplication. The core idea: \(a^m\) means \(a\) multiplied by itself \(m\) times.

Key principle: All rules follow logically from the basic definition. Understanding the derivation lets you reconstruct them under exam pressure.

The Seven Rules

RuleFormulaDerivation / Notes
Rule 1\(a^m \times a^n = a^{m+n}\)\(m\) copies then \(n\) copies gives \(m+n\) copies
Rule 2\(a^{1/n} = \sqrt[n]{a}\)Multiplying \(n\) times: \((a^{1/n})^n = a^1 = a\), so it is the \(n\)th root
Rule 3\(a^{-m} = \dfrac{1}{a^m}\)\(a^3 \times a^{-2} = a^1\), so \(a^{-2} = 1/a^2\)
Rule 4\(a^0 = 1\)\(a^2 \times a^{-2} = a^0\) and \(a^2/a^2 = 1\), so \(a^0 = 1\)
Rule 5\(a^m \div a^n = a^{m-n}\)Combine Rules 1 & 3: \(a^m \times a^{-n} = a^{m-n}\)
Rule 6\((a^m)^n = a^{mn}\)\(n\) copies of \(a^m\) → exponent \(m+m+\cdots+m = mn\)
Rule 7\(a^{m/n} = \sqrt[n]{a^m} = \!\left(\sqrt[n]{a}\right)^m\)Combine Rules 2 & 6
⚠ Critical Distinction

Do not confuse \((a^3)^2 = a^6\) with \(a^{3^2} = a^9\). They look similar but mean completely different things.

Validity Conditions

All rules require \(a > 0\) for full generality. Negative bases cause problems: \((-64)^{1/2}\) is not real, but \((-64)^{1/3} = -4\). When \(a = 0\): \(0^m = 0\) for \(m > 0\), but \(0^0\) is undefined.

Worked Examples

Example A — Solve using index laws

Solve \(\dfrac{27^a}{3^{a-1}} = 3\sqrt{3}\)

\(27^a = (3^3)^a = 3^{3a}\)
\(\dfrac{3^{3a}}{3^{a-1}} = 3^{3a-(a-1)} = 3^{2a+1}\)
\(3\sqrt{3} = 3^{3/2} \;\Rightarrow\; 2a+1 = \tfrac{3}{2} \;\Rightarrow\; \boxed{a = \tfrac{1}{4}}\)
Example B — Substitution approach

Solve \(4^x - 2^{x+2} = 32\)

Let \(u = 2^x\). Then \(4^x = u^2\) and \(2^{x+2} = 4u\).
\(u^2 - 4u - 32 = 0 \;\Rightarrow\; (u-8)(u+4) = 0\)
\(u = 8 \Rightarrow 2^x = 8 \Rightarrow \boxed{x = 3}\)   (reject \(u = -4\))
Practice Laws of Indices Questions
1
Solve the equation  \(2^{x+2} = 4\sqrt{2}\)
Easy
Solution

Write both sides as powers of 2:

\(4\sqrt{2} = 2^2 \cdot 2^{1/2} = 2^{5/2}\) \(2^{x+2} = 2^{5/2} \;\Rightarrow\; x + 2 = \dfrac{5}{2}\)
Answer: \(x = \dfrac{1}{2}\)
2
Solve \(4^x - 2^{x+2} = 32\)
Easy
Solution

Let \(u = 2^x\), so \(4^x = u^2\) and \(2^{x+2} = 4u\):

\(u^2 - 4u - 32 = 0 \;\Rightarrow\; (u - 8)(u + 4) = 0\) \(u = 8 \Rightarrow 2^x = 8\)   (reject \(u = -4\), no real solution)
Answer: \(x = 3\)
3
How many real solutions does \(8^x + 4 = 4^x + 2^{x+2}\) have?
Medium
Solution

Let \(u = 2^x\). Rearranging gives \(u^3 - u^2 - 4u + 4 = 0\):

\((u^2 - 4)(u - 1) = 0 \;\Rightarrow\; u = 1,\; 2,\; -2\) \(2^x = 1 \Rightarrow x=0\);   \(2^x = 2 \Rightarrow x=1\);   \(2^x = -2\) — no real solution
Answer: 2 real solutions  (\(x = 0\) and \(x = 1\))
4
Find the values of \(k\) such that \(9^x - 3^{x+1} = k\) has one or more real solutions.
Medium
Solution

Let \(t = 3^x > 0\). The equation becomes \(t^2 - 3t - k = 0\), a quadratic in \(t\).

For a real positive solution, the discriminant must be \(\geq 0\):

\(9 + 4k \geq 0 \;\Rightarrow\; k \geq -\tfrac{9}{4}\)
Answer: \(k \geq -\dfrac{9}{4}\)
5
Solve \(\dfrac{27^a}{3^{a-1}} = 3\sqrt{3}\)
Easy
Solution
\(27^a = 3^{3a},\quad 3\sqrt{3} = 3^{3/2}\) \(\dfrac{3^{3a}}{3^{a-1}} = 3^{2a+1} = 3^{3/2}\) \(2a + 1 = \tfrac{3}{2} \;\Rightarrow\; 2a = \tfrac{1}{2}\)
Answer: \(a = \dfrac{1}{4}\)
✦
MM1.2

Surds

A surd is an expression containing a root that cannot be simplified to a rational number. Surds let us express irrational numbers exactly.

Convention

\(\sqrt{a}\) always denotes the positive square root. So \(\sqrt{64} = 8\), not \(-8\). Write \(\pm\sqrt{64}\) if both signs are needed.

1 · Simplifying Roots

Factor out the largest perfect square from under the root sign:

\(\sqrt{50} = \sqrt{25 \times 2} = 5\sqrt{2}\)
\(\sqrt{40} = \sqrt{4 \times 10} = 2\sqrt{10}\)
\(\sqrt{72} = \sqrt{36 \times 2} = 6\sqrt{2}\)

2 · Multiplying Surd Expressions

Treat \(\sqrt{5}\) like a variable \(x\), expand normally, then use \(\sqrt{5}\times\sqrt{5}=5\) at the end:

Example — Expand and Simplify
\((2 + 3\sqrt{5})^2 = 4 + 2 \cdot 2 \cdot 3\sqrt{5} + 9 \cdot 5 = 4 + 12\sqrt{5} + 45 = 49 + 12\sqrt{5}\)

3 · Rationalising the Denominator

Simple: \(1/\sqrt{a}\)

\(\dfrac{1}{\sqrt{a}} \times \dfrac{\sqrt{a}}{\sqrt{a}} = \dfrac{\sqrt{a}}{a}\)

Conjugate: \((a+b)(a-b)=a^2-b^2\)

\(\dfrac{3}{2+\sqrt{5}} \times \dfrac{\sqrt{5}-2}{\sqrt{5}-2} = \dfrac{3\sqrt{5}-6}{5-4} = 3\sqrt{5}-6\)

⚠ Common Mistake

Never write \(\sqrt{5} \times \sqrt{5} = 25\). Always: \(\sqrt{5} \times \sqrt{5} = 5\).

Practice Surds Questions
1
Simplify \(\dfrac{(2\sqrt{3}-1)(2+\sqrt{3})}{(2-\sqrt{3})(2+\sqrt{3})}\) in the form \(a + b\sqrt{3}\).
Easy
Solution
Denominator: \((2-\sqrt{3})(2+\sqrt{3}) = 4 - 3 = 1\) Numerator: \((2\sqrt{3}-1)(2+\sqrt{3}) = 4\sqrt{3} + 6 - 2 - \sqrt{3} = 4 + 3\sqrt{3}\)
Answer: \(4 + 3\sqrt{3}\)
2
The area of a triangle is \((3 + \sqrt{3})\) cm². The base is \(\sqrt{3}\) cm. Find the height as a surd.
Medium
Solution
\(\tfrac{1}{2}\sqrt{3} \cdot h = 3 + \sqrt{3} \;\Rightarrow\; h = \dfrac{2(3+\sqrt{3})}{\sqrt{3}}\) \(= \dfrac{(6 + 2\sqrt{3})\sqrt{3}}{3} = \dfrac{6\sqrt{3} + 6}{3} = 2\sqrt{3} + 2\)
Answer: \(h = 2\sqrt{3} + 2\)
3
What positive integer does \(\dfrac{1+\sqrt{7}}{3-\sqrt{7}} - \dfrac{8-\sqrt{7}}{\sqrt{7}-2}\) simplify to?
Medium
Solution

Rationalise each fraction using the conjugate, then combine:

\(\dfrac{(1+\sqrt{7})(\sqrt{7}-2) - (8-\sqrt{7})(3-\sqrt{7})}{(3-\sqrt{7})(\sqrt{7}-2)}\) Numerator \(= 10\sqrt{7} - 26\),   Denominator \(= 5\sqrt{7}-13 = \sqrt{7}(5\sqrt{7}-13)\cdot\tfrac{1}{\sqrt{7}}\) \(\dfrac{10\sqrt{7}-26}{5\sqrt{7}-13} = \dfrac{2(5\sqrt{7}-13)}{5\sqrt{7}-13} = 2\)
Answer: \(2\)
4
Write \(\dfrac{1}{x-\sqrt{y}} + \dfrac{1}{x+\sqrt{y}}\) as a single fraction in simplest form.
Easy
Solution
\(\dfrac{(x+\sqrt{y}) + (x-\sqrt{y})}{(x-\sqrt{y})(x+\sqrt{y})} = \dfrac{2x}{x^2 - y}\)
Answer: \(\dfrac{2x}{x^2 - y}\)
5
What is \(\sqrt{49 - 12\sqrt{5}}\)?   (The answer is not \(2 - 3\sqrt{5}\).
Hard
Solution

Try \((a - b\sqrt{5})^2 = a^2 + 5b^2 - 2ab\sqrt{5}\). Need \(2ab = 12\) and \(a^2 + 5b^2 = 49\).

\(ab = 6\);   try \(a=2, b=3\): \(4 + 45 = 49\) ✓ So \(49 - 12\sqrt{5} = (2 - 3\sqrt{5})^2\) — but \(2 - 3\sqrt{5} < 0\)! Since \(\sqrt{\cdot}\) is always positive: \(\sqrt{49-12\sqrt{5}} = |2-3\sqrt{5}| = 3\sqrt{5}-2\)
Answer: \(3\sqrt{5} - 2\)
✦
MM1.3

Quadratic Functions & the Discriminant

A quadratic has the form \(ax^2 + bx + c\) where \(a \neq 0\). Mastery of all aspects below is essential.

Completing the Square

General Form (\(a = 1\))

\[x^2 + bx + c = \left(x + \frac{b}{2}\right)^2 - \frac{b^2}{4} + c\]

The graph is \(y = x^2\) shifted left by \(b/2\) and up by \(c - b^2/4\). Minimum at \(\!\left(-\tfrac{b}{2},\; c - \tfrac{b^2}{4}\right)\).

General Form (any \(a\))

\[ax^2 + bx + c = a\!\left(x + \frac{b}{2a}\right)^2 - \frac{b^2 - 4ac}{4a}\]

The Quadratic Formula

\[x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\]

The axis of symmetry is \(x = -b/2a\). Both roots lie at distance \(\dfrac{\sqrt{b^2-4ac}}{2|a|}\) from it.

The Discriminant \(\Delta = b^2 - 4ac\)

ConditionRootsGraph behaviour
\(\Delta > 0\)Two distinct real rootsParabola cuts \(x\)-axis at two points
\(\Delta = 0\)One repeated rootParabola touches \(x\)-axis at vertex
\(\Delta < 0\)No real rootsParabola entirely above/below \(x\)-axis
Practice Quadratics Questions
1
\(x^2 + ax + b = 0\) has roots \(x = -2\) and \(x = 5\). Find \(a\) and \(b\).
Easy
Solution
\((x+2)(x-5) = x^2 - 3x - 10\) Comparing: \(a = -3,\quad b = -10\)
Answer: \(a = -3,\quad b = -10\)
2
\(f(x)=ax^2+bx+c\) with \(f(-1)=f(5)=30\) and minimum value \(-6\). Solve \(f(x)=3\).
Hard
Solution

By symmetry the vertex is at \(x = \tfrac{-1+5}{2} = 2\), so \(f(x) = k(x-2)^2 - 6\).

Using \(f(5)=30\): \(9k-6=30 \Rightarrow k=4\) \(f(x) = 4(x-2)^2 - 6 = 3 \;\Rightarrow\; (x-2)^2 = \tfrac{9}{4} \;\Rightarrow\; x-2 = \pm\tfrac{3}{2}\)
Answer: \(x = \dfrac{1}{2}\) or \(x = \dfrac{7}{2}\)
3
Solve \(x - \dfrac{14}{x} = 6\sqrt{2}\), giving answers in the form \(p\sqrt{2}\).
Medium
Solution
Multiply by \(x\): \(x^2 - 6\sqrt{2}\,x - 14 = 0\) \(x = \dfrac{6\sqrt{2} \pm \sqrt{72 + 56}}{2} = \dfrac{6\sqrt{2} \pm \sqrt{128}}{2} = \dfrac{6\sqrt{2} \pm 8\sqrt{2}}{2}\)
Answer: \(x = 7\sqrt{2}\) or \(x = -\sqrt{2}\)
4
Given \(f(x)=x^2+10x+27\), find \(k\) such that \(f(x)-k\) touches the \(x\)-axis.
Easy
Solution
Complete the square: \(f(x) = (x+5)^2 + 2\), so minimum value \(= 2\) \(f(x) - k\) touches \(x\)-axis when minimum \(= 0\):   \(2 - k = 0\)
Answer: \(k = 2\)
5
A quadratic passes through \((-2,0)\), \((4,0)\), and \((0,-20)\). Find its equation.
Medium
Solution
\(y = a(x+2)(x-4)\). At \((0,-20)\): \(-8a = -20 \Rightarrow a = \tfrac{5}{2}\)
Answer: \(y = \dfrac{5}{2}x^2 - 5x - 20\)
6
Find values of \(k\) so that \(y = k(2x^2+1)\) and \(y = x^2-2x\) touch each other.
Hard
Solution

Set equal and collect: \(x^2(2k-1) + 2x + k = 0\). For tangency, \(\Delta = 0\):

\(4 - 4k(2k-1) = 0 \;\Rightarrow\; 2k^2 - k - 1 = 0\) \((2k+1)(k-1) = 0\)
Answer: \(k = -\dfrac{1}{2}\) or \(k = 1\)
7
\(f(n) = n^2 - 2kn + k + 12\) is a perfect square for all \(n \in \mathbb{N}\). Find possible values of \(k\).
Hard
Solution

Complete the square: \(f(n) = (n-k)^2 - k^2 + k + 12\). For this to be a perfect square for all \(n\), the constant term must vanish:

\(-k^2 + k + 12 = 0 \;\Rightarrow\; k^2 - k - 12 = 0\) \((k-4)(k+3) = 0\)
Answer: \(k = 4\) or \(k = -3\)
✦
MM1.4

Simultaneous Equations

Solving simultaneously means finding values satisfying both equations. Geometrically: solutions are intersection points of graphs.

Two Linear Equations — Three Cases

GeometrySolutionsExample
Parallel, distinctNone\(y+2x=4\) and \(y+2x=8\)
Coincident (same line)Infinitely many\(y+2x=4\) and \(2y+4x=8\)
Non-parallelExactly 1\(x+2y=5\) and \(2x+y=4\)

Linear + Quadratic

\(\Delta\) of resulting quadraticGeometric meaning
\(\Delta > 0\)Line crosses parabola — 2 intersections
\(\Delta = 0\)Line is tangent — 1 (repeated) intersection
\(\Delta < 0\)Line misses parabola — no real intersections
Practice Simultaneous Equations Questions
1
Solve simultaneously: \(x + 2y = 5\) and \(2x + y = 4\).
Easy
Solution
From first equation: \(x = 5 - 2y\) Substitute: \(2(5-2y) + y = 4 \;\Rightarrow\; 10 - 3y = 4 \;\Rightarrow\; y = 2\) Then \(x = 5 - 4 = 1\)
Answer: \(x = 1,\quad y = 2\)
2
The straight line \(L\) passes through \((0,-1)\). The curve \(y = x^2+2x\) does not intersect \(L\). Find the range of possible gradients of \(L\).
Hard
Solution

Line: \(y = mx - 1\). Set equal: \(x^2 + 2x = mx - 1 \;\Rightarrow\; x^2 + (2-m)x + 1 = 0\).

No intersections requires \(\Delta < 0\):

\((2-m)^2 - 4 < 0 \;\Rightarrow\; m^2 - 4m < 0 \;\Rightarrow\; m(m-4) < 0\)
Answer: \(0 < m < 4\)
3
Find the range of \(k\) so that \(y = 4x^2-7x+11\) and \(y = 5x+k\) do not intersect.
Medium
Solution
Set equal: \(4x^2 - 12x + 11 - k = 0\). Need \(\Delta < 0\). \(144 - 16(11-k) < 0 \;\Rightarrow\; -32 + 16k < 0 \;\Rightarrow\; k < 2\)
Answer: \(k < 2\)
✦
MM1.5

Inequalities

⚠ Golden Rules

You can add/subtract freely on both sides. Multiplying or dividing by a negative flips the inequality sign. Squaring both sides can introduce false solutions — always verify.

Modulus — Geometric Insight

\(|x - a|\) measures the distance of \(x\) from \(a\) on the number line. So \(|x-3| < |x-5|\) asks: "for which \(x\) is the distance to 3 less than to 5?" The midpoint of 3 and 5 is 4, so \(x < 4\).

Practice Inequalities Questions
1
Solve \(x^4 < 8x^2 + 9\).
Medium
Solution
\(x^4 - 8x^2 - 9 < 0 \;\Rightarrow\; (x^2 - 9)(x^2 + 1) < 0\) Since \(x^2 + 1 > 0\) always, need \(x^2 - 9 < 0 \;\Rightarrow\; -3 < x < 3\)
Answer: \(-3 < x < 3\)
2
Solve the inequality \(12 - 2|2x - 3| > 7\).
Easy
Solution
\(2|2x-3| < 5 \;\Rightarrow\; |2x-3| < \tfrac{5}{2}\) \(-\tfrac{5}{2} < 2x-3 < \tfrac{5}{2} \;\Rightarrow\; \tfrac{1}{4} < x < \tfrac{11}{4}\)
Answer: \(\dfrac{1}{4} < x < \dfrac{11}{4}\)
3
Solve the equation \(|3x + 2| = 1\).
Easy
Solution
Case 1: \(3x+2 = 1 \;\Rightarrow\; x = -\tfrac{1}{3}\) Case 2: \(3x+2 = -1 \;\Rightarrow\; x = -1\)
Answer: \(x = -\dfrac{1}{3}\) or \(x = -1\)
4
Solve \(|x^2 - 2x - 4| = 4\).
Medium
Solution
Case 1: \(x^2-2x-4 = 4 \;\Rightarrow\; (x-4)(x+2)=0 \;\Rightarrow\; x=4,\,-2\) Case 2: \(x^2-2x-4 = -4 \;\Rightarrow\; x(x-2)=0 \;\Rightarrow\; x=0,\,2\)
Answer: \(x = -2,\;0,\;2,\;4\)
5
Find the set of values for which \(|x^2 - 4| < 3x\).
Hard
Solution

Note \(3x > 0\) is required, so \(x > 0\). Solve both cases:

Case 1: \(x^2-4 = 3x \;\Rightarrow\; (x-4)(x+1)=0 \;\Rightarrow\; x=4\) (take \(x>0\)) Case 2: \(x^2-4 = -3x \;\Rightarrow\; (x+4)(x-1)=0 \;\Rightarrow\; x=1\) (take \(x>0\)) Sketch confirms inequality holds between: \(1 < x < 4\)
Answer: \(1 < x < 4\)
6
\(f(x) = x+4\), \(g(x) = |2x+1|+3\). Solve \(f(g(x)) > 12\).
Medium
Solution
\(f(g(x)) = |2x+1| + 3 + 4 = |2x+1| + 7 > 12\) \(|2x+1| > 5 \;\Rightarrow\; 2x+1 > 5\) or \(2x+1 < -5\) \(x > 2\)   or   \(x < -3\)
Answer: \(x < -3\) or \(x > 2\)
✦
MM1.6

Polynomials, Factor & Remainder Theorems

Factor Theorem (if and only if)

For polynomial \(f(x)\):   \(f(a) = 0 \;\Longleftrightarrow\; (x-a)\) is a factor of \(f(x)\)

Remainder Theorem

When \(f(x)\) is divided by \((x-b)\), the remainder is \(f(b)\).
When divided by \((px-q)\), the remainder is \(f(q/p)\).
The Factor Theorem is the special case where remainder \(= 0\).

Practice Polynomials Questions
1
\(x^3 + 4x^2 + 7x + a\) has factor \((x+2)\). Find \(a\).
Easy
Solution
\(f(-2) = 0\!: \quad -8 + 16 - 14 + a = 0 \;\Rightarrow\; a = 6\)
Answer: \(a = 6\)
2
For \(f(x)=ax^3-x^2-5x+b\): remainder 36 when divided by \((x-2)\), and 40 by \((x+2)\). Find \(a\) and \(b\).
Medium
Solution
\(f(2) = 8a - 4 - 10 + b = 36 \;\Rightarrow\; 8a + b = 50 \quad\cdots①\) \(f(-2) = -8a - 4 + 10 + b = 40 \;\Rightarrow\; -8a + b = 34 \quad\cdots②\) ①\(-\)②: \(\;16a = 16 \Rightarrow a = 1\);   then \(b = 42\)
Answer: \(a = 1,\quad b = 42\)
3
\(f(x)=6x^2+x+7\). Remainder from \((x-a)\) equals remainder from \((x+2a)\), \(a\neq 0\). Find \(a\).
Medium
Solution
\(f(a) = f(-2a)\!: \quad 6a^2 + a + 7 = 24a^2 - 2a + 7\) \(0 = 18a^2 - 3a = 3a(6a-1)\) \(a = 0\) (rejected)   or   \(a = \tfrac{1}{6}\)
Answer: \(a = \dfrac{1}{6}\)
4
\(g(x)=x^3+kx^2-x+12\). Remainder when divided by \((x-4)\) is 8 times the remainder from \((x-1)\). Find \(k\).
Medium
Solution
\(g(4) = 64 + 16k - 4 + 12 = 72 + 16k\) \(g(1) = 1 + k - 1 + 12 = k + 12\) \(72 + 16k = 8(k+12) \;\Rightarrow\; 8k = 24 \;\Rightarrow\; k = 3\)
Answer: \(k = 3\)
5
Find the remainder when \(1 + 3x + 5x^2 + 7x^3 + \cdots + 99x^{49}\) is divided by \((x-1)\).
Hard
Solution

By the Remainder Theorem, substitute \(x = 1\):

\(f(1) = 1 + 3 + 5 + 7 + \cdots + 99\) Sum of 50 consecutive odd numbers starting from 1: \(= 50^2 = 2500\)
Answer: Remainder \(= 2500\)
✦
MM1.7

Functions

A function maps each input to exactly one output. Three requirements: a rule, a domain, and the single-output guarantee.

Vertical & Horizontal Line Tests

Function test: Every vertical line through the graph crosses it at most once.
One-to-one: Every horizontal line also crosses at most once.
Many-to-one: Some horizontal line crosses more than once (e.g. \(y = x^2\)).

FunctionDomainTypeKey property
\(f(x)=x^2\)All realsMany-to-one\(f(2)=f(-2)=4\)
\(f(x)=x^3\)All realsOne-to-oneEach output unique
\(f(x)=\sqrt{x}\)\(x\geq 0\)One-to-oneAlways positive root
\(f(x)=|x|\)All realsMany-to-one\(|-3|=|3|=3\)
Practice Functions Questions
1
Classify each: (a) \(f(x)=x^2\)   (b) \(f(x)=\pm\sqrt{x}\)   (c) \(f(x)=x^3\)   (d) \(x^2+y^2=9\)
Easy
Solution
(a) \(f(x)=x^2\): Function — many-to-one (e.g. \(f(2)=f(-2)=4\)) (b) \(\pm\sqrt{x}\): Not a function — two outputs per input (fails vertical line test) (c) \(f(x)=x^3\): Function — one-to-one (d) \(x^2+y^2=9\): Not a function — circle fails vertical line test
2
\(f(1)=5\); \(f(n+1)=3f(n)-1\) if \(f(n)\) is odd; \(f(n+1)=\tfrac{f(n)}{2}\) if even. Find \(f(99)\) and how many \(n \in [1,50]\) satisfy \(f(n) \leq 12\).
Hard
Solution

Compute the sequence to find the period:

\(f(1)=5,\; f(2)=14,\; f(3)=7,\; f(4)=20,\; f(5)=10,\; f(6)=5,\ldots\) Period is 5. Since \(99 = 5 \times 19 + 4\),   \(f(99) = f(4) = 20\) Values \(\leq 12\) in one period: \(f(2)=14>12\) ✗, \(f(3)=7\) ✓, \(f(5)=10\) ✓, \(f(6)=5\) ✓ ... 3 out of 5 per period In 50 terms: \(\tfrac{3}{5} \times 50 = 30\)
\(f(99) = 20\);   30 values satisfy \(f(n) \leq 12\)
3
\(f(mn)=f(m)f(n)\) if \(mn\) is a multiple of 5, else \(f(mn)=mn\). Given \(f(25)+f(9)-f(30)=0\), find \(f(5)\).
Hard
Solution
\(f(25) = [f(5)]^2\),   \(f(9) = 9\),   \(f(30) = f(5)\cdot f(6) = 6f(5)\) \([f(5)]^2 + 9 - 6f(5) = 0\) Let \(y = f(5)\): \(y^2 - 6y + 9 = 0 \;\Rightarrow\; (y-3)^2 = 0\)
Answer: \(f(5) = 3\)
✦
Mixed

Discriminant Applications

Finding parameter ranges using the discriminant — a core TMUA/ESAT question type.

Practice Discriminant Range Questions
1
\(x^2 + 3ax + a = 0\) has two distinct real roots. Find the range of \(a\).
Medium
Solution
\(\Delta > 0\!: \quad 9a^2 - 4a > 0 \;\Rightarrow\; a(9a-4) > 0\)
Answer: \(a < 0\) or \(a > \dfrac{4}{9}\)
2
\(x^2 + (k-1)x + (k+2) = 0\) has no real roots. Find the range of \(k\).
Medium
Solution
\(\Delta < 0\!: \quad (k-1)^2 - 4(k+2) < 0\) \(k^2 - 6k - 7 < 0 \;\Rightarrow\; (k-7)(k+1) < 0\)
Answer: \(-1 < k < 7\)
3
\(2x^2 + (3k-1)x + (3k^2-1) = 0\) has two different real roots. Find the range of \(k\).
Hard
Solution
\(\Delta > 0\!: \quad (3k-1)^2 - 8(3k^2-1) > 0\) \(-15k^2 - 6k + 9 > 0 \;\Rightarrow\; 5k^2 + 2k - 3 < 0\) \((5k-3)(k+1) < 0\)
Answer: \(-1 < k < \dfrac{3}{5}\)
4
\(kx^2 - x + (3k-1) = 0\) (\(k \neq 0\)) has no real roots. Find the range of \(k\).
Hard
Solution
\(\Delta < 0\!: \quad 1 - 4k(3k-1) < 0\) \(12k^2 - 4k - 1 > 0 \;\Rightarrow\; (6k+1)(2k-1) > 0\)
Answer: \(k < -\dfrac{1}{6}\) or \(k > \dfrac{1}{2}\)
5
The roots of \(2x^2 - 7x + c = 0\) differ by 3. Find \(c\).
Hard
Solution

Difference of roots \(= \dfrac{\sqrt{\Delta}}{a}\). So \(\dfrac{\sqrt{49 - 8c}}{2} = 3\):

\(\sqrt{49-8c} = 6 \;\Rightarrow\; 49 - 8c = 36 \;\Rightarrow\; 8c = 13\)
Answer: \(c = \dfrac{13}{8}\)
